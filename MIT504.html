Ders Amacı

Temel internet ve web geliştirme teknolojilerini tanıtma, HTML ve web programlama temel yeteneklerinin geliştirilmesi.

Ders Programı

  1. Web programlamaya giriş, web geliştirme teknolojileri (ders notları).
  2. Web sayfasının temel yapısı, HTML dili (ders notları).
  3. Stiller ve web sayfasının biçimlendirmesi (ders notları).
  4. Basamaklı Stil Sayfalarına (CSS) giriş (ders notları).
  5. Şablonlar kullanarak özel bir web sitesinin geliştirilmesi (ders notları).
  6. HTML ve CSS ile pratik çalışma
  7. Javascript'e giriş, Dinamik HTML (ders notları).
  8. Javascript'in temelleri
  9. Javascript'i kullanarak dinamik web sayfasının geliştirilmesi
  10. Javascript'i kullanarak kullanıcı ile etkileşim
  11. Dinamik HTML ve Javascript ile pratik çalışma
  12. Final proje çalışması
