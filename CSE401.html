CSE401 Information Systems Security

Instructor:Assoc. Prof. Dr. Yuriy Mishchenko (Toros University)
Email:yuriy.mishchenko@toros.edu.tr

Resources

  • Textbook: Textbook and reading materials will be given in the class.

Objective

Survey of principles and issues of the security of modern information systems, encryption theory, security policies, vulnerabilities and information security attacks.

Program

  1. Subject and topic of Information Security
  2. Overview of basic concepts in Information Security
  3. Access Control I (Domain 1)
  4. Access Control II
  5. Telecommunications and Network Security I (Domain 2)
  6. Telecommunications and Network Security II
  7. MIDTERM
  8. Software development security (Domain 4)
  9. Cryptography and encryption I (Domain 5)
  10. Cryptography and encryption II
  11. Security architecture and design (Domain 6)
  12. Security operations (Domain 7)
  13. Information security governance and risk management (Domain 3)
  14. Business continuity and recovery planning (Domain 8)
